🐍 Python Mini Projects

A collection of beginner-friendly Python projects built to practice core programming concepts like loops, functions, conditionals, and user input.


📁 Projects

1. 🧮 Calculator (calculator.py)

A simple command-line calculator that supports basic arithmetic operations.

Features:


Addition, Subtraction, Multiplication, Division
Continuously runs until the user chooses to exit


2. 🔄 Value Converter (convert.py)

A multi-purpose unit converter with three conversion categories.

Features:


Temperature — Celsius ↔ Fahrenheit
Currency — US Dollars ↔ British Pounds
Length — Centimeters ↔ Feet & Inches



3. 🎲 Number Guessing Game (import_random.py)

A fun guessing game where the player tries to guess a randomly generated number between 0 and 9.

Features:


Random number generation using Python's random module
Hints to guide the player (higher/lower, close/not close)
Tracks the number of attempts


4. 🔐 Password Strength Checker (password_strength.py)

A tool that evaluates the strength of a password based on its character composition.

Features:


Checks for lowercase letters, uppercase letters, digits, spaces, and special characters
Scores the password out of 5
Gives remarks based on the strength
Uses getpass to hide the password while typing


🚀 Getting Started

Make sure you have Python 3 installed. You can download it from python.org.

Clone or download this repository and run any project using:

bashpython <filename>.py


👨‍💻 About

These are beginner-level Python projects made by me as part of learning and practising Python programming. Each project focuses on a simple concept and runs entirely in the terminal.

Python 3.x
